# sphere. | KI-Automation für Fitnessstudios

Dieses Projekt ist eine moderne Landingpage für **sphere**, spezialisiert auf KI-Implementierung und Automationen für Fitnessstudios.

## Technologien
- HTML, Vanilla CSS, Javascript
- [Vite](https://vitejs.dev/) als Build Tool
- [Vercel](https://vercel.com/) für Deployment und Hosting

## Lokale Entwicklung
1. Abhängigkeiten installieren: `npm install`
2. Entwicklungsserver starten: `npm run dev`
3. Produktionsbuild erstellen: `npm run build`

## Deployment auf Vercel
1. Den Inhalt dieses Ordners in ein neues GitHub Repository pushen.
2. In Vercel ein neues Projekt importieren und das GitHub Repo auswählen.
3. Die standardmäßigen Vite-Einstellungen werden von Vercel automatisch erkannt.
